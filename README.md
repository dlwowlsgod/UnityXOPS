# UnityXOPS
UnityXOPS는 TEAM-MITEI(nine-two, TENNKUU)가 만든 프리웨어 FPS [X Operation](https://rucsgames.net/se/xops/)을 Unity로 구현한 오픈소스 게임입니다.
## Installation
1. Release에 있는 게임 파일을 원하는 버전으로 받습니다.
2. 원본 X Operation을 다운받고, data 폴더를 UnityXOPS\UnityXOPS_Data\StreamingAssets 폴더에 넣습니다.
  > [원본 다운로드](https://www.vector.co.jp/soft/win95/game/se284467.html) | [네이버 카페](https://cafe.naver.com/xopertaion)
3. UnityXOPS.exe를 실행합니다.
