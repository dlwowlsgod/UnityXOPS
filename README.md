# UnityXOPS

Editor Version : 6000.0.77f

Installation :
- Download latest (or specific)release UnityXOPS
- Extract, and copy original asset data (data folder) to UnityXOPS_Data/StreamingAssets folder

Features :
- json extracted datas (Human, Weapon, Object, etc...)
- F1~F3 to change view (3rd left, 1st, 3rd right)
- F4 to change UI (normal, simple)

Restrictions :
- Cheat not implemented (~0.1) (forgotten)
- 3rd Camera is different as Original XOPS
- Using Error Human, Weapon addon mission is not work properly
