# UnityXOPS

Editor Version : 6000.0.77f

Installation :
- Download latest (or specific)release UnityXOPS
- Download Unity Hub, Install apprepriated Editor version
- Extract all source code to Project folder
- Extract and copy original XOPS asset data (data folder) to UnityXOPS_Data/StreamingAssets folder

Features :
- json extracted datas (Human, Weapon, Object, etc...)
- lua scripting (hot reload, no compile needed feature edits, etc...)

Restrictions :
- Using Error Human, Weapon addon mission is not work properly
