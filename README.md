# UnityXOPS
## Installation
>   1. Download and install [Unity 6000.0f](https://unity.com/download)
>   2. Create project and import everything
>   3. Import TextMeshPro essential resources
>   4. Install [UniTask](https://github.com/Cysharp/UniTask) and [Assimp for Unity](https://intelligide.github.io/assimp-unity/)
## Build Settings
>  1. Add all scenes (Init, Opening, Mainmenu, Briefing, Game, Result) to Build scenes
